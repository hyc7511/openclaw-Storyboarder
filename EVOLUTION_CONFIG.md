## 二、问题反馈流程

### 2.1 反馈接收方式

用户可通过以下方式提供反馈：

1. **直接指出问题**: "这里有个问题：{问题描述}"
2. **标准反馈格式**:

```
问题描述：{详细描述}
问题出现环节：{导演分析/服化道设计/分镜设计}
建议改进方向：{可选}
```

### 2.2 问题分析与责任判定

系统将自动：

1. 识别问题类型（匹配已知问题库或分类新问题）
2. 判定责任环节（哪个Agent负责）
3. 确定需要强化审核的阶段
4. 更新演进记录

***

## 三、强制审核机制

### 3.1 问题触发的强化审核

当识别到特定问题后，在后续迭代中强制执行：

```
例如：识别到"角色越轴"问题
→ 责任判定：分镜师
→ 强化点：在分镜设计阶段强制增加"180度准则检查"
→ 在导演审核阶段强制增加"镜头语言专项审核"
```

### 3.2 审核清单自动更新

每个已识别问题都会自动附加到对应阶段的审核清单中，成为必检项。
