# Video Production Skill

## Skill Name

视频制作流程管理

## Description

负责协调整个视频提示词生成的三阶段流程，包括任务分配、进度追踪和状态管理。

## Capabilities

- Workflow orchestration (3-phase process)
- Task assignment to specialized agents
- Progress tracking and state persistence
- Quality gate management
- User guidance through the process

## Usage

This skill is activated automatically when the user provides a script or requests to start the video production process.

## Workflow Phases

1. **Director Analysis**: Script breakdown and creative analysis
2. **Art Design**: Character and scene visual design
3. **Storyboarding**: Seedance 2.0 prompt generation

## State Management

The skill maintains state in `.agent-state.json` to support resumable workflows.
