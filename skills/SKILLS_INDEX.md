# 技能索引 - 影视制作工作区

## 技能概览

本工作区包含10个核心技能模块，涵盖影视制作的各个环节，确保从剧本分析到最终分镜输出的全流程专业质量。

---

## 导演技能组

### 1. 导演技能
**路径**: `skills/director-skill/SKILL.md`

**核心职能**:
- 剧本深度分析与解读
- **导演解场与讲戏**（焦点人物提取、动作拆解、情绪基调确定）

## 镜头类型参考

| 类型 | 英文 | 特点与用途 |
|------|------|------------|
| 远景/全景 | wide | 展示环境，建立场景氛围 |
| 中景 | medium | 展示人物上半身和动作 |
| 中近景 | medium close-up | 人物腰部以上，兼顾动作与表情 |
| 近景/特写 | close-up | 人物胸部以上，强调表情 |
| 大特写 | extreme close-up | 细节特写，强调重点 |

## 镜头运动参考

| 运动类型 | 英文 | 效果与用途 |
|----------|------|------------|
| 固定 | static | 稳定观察，客观呈现 |
| 推近 | dolly_in | 引导注意力，强调重点 |
| 拉远 | dolly_out | 展示环境，揭示场景 |
| 水平摇移 | pan | 水平环顾，跟随动作 |
| 垂直摇移 | tilt | 垂直移动，展示上下 |
| 追踪 | tracking | 跟随人物，保持关注 |
| 手持 | handheld | 增加真实感与紧张感 |

## Seedance 提示词规范

### 组合公式
```
[环境基础] + [主体及动作（加权重）] + [镜头语言] + [画质修饰]
```

### 强制画风（真人漫剧）
```
masterpiece, best quality, ultra-detailed, realistic photography, 8K, cinematic lighting, shallow depth of field
```

### 负面词库
```
(worst quality, low quality:1.4), (deformed, distorted, disfigured:1.3), poorly drawn, bad anatomy, wrong anatomy, extra limb, missing limb, floating limbs, mutated hands and fingers, cartoon, anime, 3d render, CG, unrealistic
```

## 技能配合工作流

### 第一阶段：剧本分析
1. **导演技能** → 进行剧本分析，生成讲戏本、人物清单、场景清单
2. **剧本分析审核技能** → 导演自审分析结果（含连贯性检查）
3. **合规审核技能** → 合规审核剧本内容（含 NSFW 检查）

### 第二阶段：服化道设计
1. **服化道设计技能** → 根据导演分析进行人物、场景设计（含 Seedance 提示词）
2. **服化道审核技能** → 导演审核服化道设计（含美术标准检查）
3. **合规审核技能** → 合规审核设计内容

### 第三阶段：分镜制作
1. **分镜师技能** → 编写Seedance 2.0分镜提示词（含编译规范）
2. **分镜审核技能** → 导演审核分镜提示词（含运镜冲突检查）
3. **合规审核技能** → 最终合规审核

## 技能使用要点

### 导演相关技能
- 使用**导演技能**进行创作
- 使用**剧本分析审核技能**进行自审
- 使用**服化道审核技能**审核服化道
- 使用**分镜审核技能**审核分镜
- 使用**合规审核技能**进行最终合规把关

### 服化道技能
- 使用**服化道设计技能**进行专业设计
- 确保输出符合JSON格式规范
- 生成标准的 Seedance 提示词
- 注重人物性格与场景氛围的体现

### 分镜师技能
- 使用**分镜师技能**进行分镜创作
- 严格遵循Seedance 2.0格式标准
- 按编译规范生成提示词
- 注重镜头语言的叙事功能

## 输出文件规范

所有技能输出都应遵循以下规范：
1. **格式统一**: 使用规定的Markdown、JSON、YAML格式
2. **内容完整**: 包含所有必需字段与信息
3. **专业规范**: 使用专业术语，表述准确清晰
4. **可追溯**: 保留版本信息与修改记录
5. **素材关联**: 正确引用人物、场景素材库资源
6. **Seedance 兼容**: 确保提示词符合 Seedance 2.0 规范
