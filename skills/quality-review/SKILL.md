# Quality Review Skill

## Skill Name

质量审核与合规检查

## Description

提供双重审核机制：业务审核（质量把控）+ 合规审核（内容安全），确保输出质量。

## Capabilities

- Business review for creative quality
- Compliance review for content safety
- Dual-review workflow management
- Revision feedback generation
- Approval gatekeeping

## Review Types

### Business Review

**Checklist**:
- [ ] Completeness: All required elements present
- [ ] Accuracy: Aligns with creative vision
- [ ] Quality: Professional and detailed
- [ ] Consistency: Maintains style across episodes
- [ ] Clarity: Easy to understand and implement

### Compliance Review

**Checklist**:
- [ ] Legal compliance
- [ ] No sensitive content
- [ ] Platform guidelines met
- [ ] Appropriate content rating
- [ ] No copyright issues

## Review Output Format

```markdown
# 审核报告 - {阶段}

## 一、业务审核
- 状态: 通过/需要修改
- 反馈意见: {详细反馈}

## 二、合规审核
- 状态: 通过/需要修改
- 反馈意见: {详细反馈}

## 三、总体结论
{审核结论和下一步建议}
```
