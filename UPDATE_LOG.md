## v2.0 - 2026-04-22

### 📁 文件分类重构：静态配置 vs 动态数据

---

### 核心变更

1. **完整的文件分类体系**
   - 📝 **静态配置（.md）** - Git管控
   - 💾 **动态数据（SQLite）** - 不纳入Git
   - 🛠️ **工具文件** - Git管控

2. **统一 SQLite 数据库**
   - `agent_state`表 - Agent状态管理
   - `issue_log`表 - 问题历史记录
   - `active_checkpoints`表 - 活跃检查点
   - `evolution_summary`表 - 统计汇总
   - 支持事务和并发锁机制

3. **删除旧 JSON 文件**
   - 删除 `project/.agent-state.json`
   - 删除 `project/evolution_log.json`
   - 保留示例 JSON 作为参考

4. **完整的迁移工具**
   - 支持从旧 JSON 迁移到 SQLite
   - 支持导出到 JSON 备份
   - 支持从 JSON 恢复
