# 影视制作工作区
## 版本: 2.1 | 更新日期: 2026-04-22

专业的影视视频提示词生成工作区，通过多Agent协作将剧本转化为高质量的Seedance 2.0动态提示词。

---

## ✨ 最新特性

### 🎉 对话式操作（v2.1）

现在，所有操作都支持自然语言命令！不需要命令行！

- 📊 **查看当前状态** - 显示整体进度
- 📈 **查看统计** - 显示统计信息  
- 🐛 **列出问题** - 显示问题列表
- 🔍 **查看检查点** - 显示检查点
- 💾 **初始化数据库** - 安全初始化
- 等等...

### 🛡️ 智能初始化（v2.1）

- 数据库不存在？自动创建！
- 数据库已存在？安全连接！
- 只创建缺失的表，不覆盖已有数据！
- 零风险初始化！

## 快速开始

### 最简单方式

1. 在OpenClaw中打开工作区
2. 直接说："开始制作" 或上传剧本文件
3. 系统会自动初始化数据库（如需要）

### 查看当前状态

在OpenClaw中说："查看当前状态"

## 主要功能

### 🎬 三阶段工作流

| 阶段 | Agent | 功能 |
|------|------|------|
| 1 | 导演 | 剧本分析、讲戏本生成、人物/场景清单 |
| 2 | 服化道 | 人物造型设计、场景设计、素材库更新 |
| 3 | 分镜师 | 分镜设计、Seedance 2.0提示词生成 |

### 🔍 双重审核机制

每个阶段都包含：
- 业务审核（导演）
- 合规审核（导演）

### 📚 跨集素材库

- 人物设定库
- 场景设定库
- 自动风格统一
- 断点续传支持

### 🎯 演进学习机制

- 问题识别
- 责任判定
- 检查点强化
- 持续学习

## 使用流程

### 准备工作

```bash

# 或命令行初始化（可选）
cd project
python init_db.py
```

1. 上传剧本文件或放到 `script/` 目录
2. 说"开始制作"
3. 跟随系统引导
